# Apresenta-o-do-Projeto-
Apresentação do projeto de  um gestor para projetos na área de eventos
Gestor de projetos - eventos 



Sistema em Java para cadastro e gerenciamento de projetos na área de eventos.



 Funcionalidades



-  Cadastro de usuários do projeto (nome, email, telefone, data nascimento)  

- Categoria do projeto (Tipos de eventos) 

- Manutenção do projeto ( data, confirmação, cancelamento, reagendamento, prazos) 

- Atividades realizadas no projeto  



O projeto será estruturado por meio da criação de classes que representam entidades do mundo real envolvidas no contexto do sistema, como usuários, projetos, equipes e tarefas.

Componentes tecnológicos

O projeto será desenvolvido em Java, utilizando um ambiente de desenvolvimento como IntelliJ ou Eclipse, com banco de dados relacional (MySQL ), rodando em um computador pessoal com sistema operacional Windows.
